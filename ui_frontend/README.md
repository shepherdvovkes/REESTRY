# UI Frontend для управления датасетами и промптами

Веб-интерфейс для управления датасетами, промптами и просмотра истории LLM обработки.

## Использование

1. Убедитесь, что backend запущен на `http://localhost:5000`
2. Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`

## Функции

### Управление промптами
- Создание, редактирование и удаление промптов
- Настройка системных и пользовательских промптов
- Привязка промптов к шагам алгоритма
- Настройка параметров (temperature, max_tokens)

### Управление датасетами
- Просмотр списка датасетов
- Создание новых датасетов
- Просмотр статистики обработки

### История обработки
- Просмотр всех вызовов LLM
- Фильтрация по шагам алгоритма и промптам
- Детальный просмотр входных/выходных данных
- Просмотр ошибок и метаданных

### Визуализация алгоритма
- Отображение всех шагов алгоритма, где используется LLM
- Статистика по каждому шагу
- Список промптов для каждого шага

## Структура файлов

- `index.html` - Основная HTML страница
- `styles.css` - Стили
- `app.js` - JavaScript логика
